# Commit
<h4>Para hacer commit:</h4>
  <p>cd la carpeta dentro (o hacer bash here en la carpeta de la solucion)</p>
  <p>git init</p>
  <p>git add .</p>
  <p>git commit -m "TITULO DE TU COMMIT"</p>
  <p>git push origin master</p>
